    • What Html Are Tags?

    ==> The Html Tags Are Composed Of Name of the element surrounded by angular brackets.
    ==> They are Classified into two types. 1) Conatiner Tags 2) Empty Tags

    ==> 1) Container Tags:

    • These Contain Other Tags or text.
    • They have Start And End Tags
    • The End Tag has a forward slash before the name.
    • Example: paragraph tag, heading tag, title tag



    ==> 2) Empty Tags:

    • These Donot Contain Other Tags or text.
    • They only have Start
    • Example: break tag, horizontal rule tag.











    Html Tags

    ==> The < html > represents the root of an HTML document.
    ==> It acts as a Container For All Other HTML Tags.
    ==> It has Two Standard Structures Head And Body.
    ==> It is a Container tag











    Body Tags

    ==> The < body > defines the documents body.
    ==> It contains all the visual contents of the webpage such as text,images,hyperlinks,tables.etc.
    ==> There Can only be one body tag in a Document.
    ==> It is a Container tag.
